# USG Agencies — Registry

US Government agencies and offices that fund work similar to DARPA BTO / BARDA: biosecurity, pandemic preparedness, infectious-disease R&D, medical countermeasures (MCM), biosurveillance, and biodefense. One markdown file per entity covering **programmes** and **grantees/awardees**.

> **Scope note:** "Find their grantees" is bounded to pandemic / biosecurity / infectious-disease / MCM work and to *notable or major* awardees — not exhaustive grant-database dumps (NIH alone makes tens of thousands of awards/year). Where a full grantee list is impractical, the file says so and lists the largest / most representative awards with sources.
